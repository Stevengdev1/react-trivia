# React + Vite

This is a React Trivia Application built with 20 questions.  It includes 2 buttons for navigating ("Previous" and "Next").
It shows one question at a time.
The user can move forward or back to see the trivia questions.
Each time the user answers the question correctly the answer is highlighted in green.
Each time the user chooses the wrong answer is highllighted in red and the correct answer is highlighted green.
If the user does not answer the question, they can see the previous or next question should they choose to but the answer options of each question will not be displayed.
During the trivia the score is being kept as they answer the questions.
After the test is done the total amount of questions answered correctly are displayed as well as their total score.
